## 🚀 المقدمة

هذا النظام هو العمود الفقري المالي لعدة مدارس. تمت ترقيته من نظام **صناديق بسيط (v2.0)** إلى **نظام محاسبي متكامل احترافي** (v3.2) مبني على **القيد المزدوج**، يشمل: فوترة وتقسيط، بوابات دفع إلكتروني، تسويات بنكية، تعدد فروع وعملات، محرك ضرائب متقدم (VAT)، سجل تدقيق شامل، إدارة ميزانيات، قيود متكررة، مذكرات ائتمان/خصم، ومراكز تكلفة.

## 🏗️ مكونات النظام (10 أقسام)

### 1️⃣ النظام الأساسي [LEGACY]

> الجداول الموروثة — مربوطة بالنظام الجديد عبر أعمدة [BRIDGE]

- **الصناديق** (`financial_funds`): 12 صندوق — فصل الأموال حسب الغرض
- **التصنيفات** (`financial_categories`): 27 تصنيف (15 إيراد + 12 مصروف)
- **الإيرادات** (`revenues`) والمصروفات (`expenses`) + **المساهمات** (`community_contributions`)

### 2️⃣ الأساس المحاسبي [ADVANCED]

- **شجرة الحسابات** (`chart_of_accounts`): 36 حساب — 5 مستويات
- **الفترات المالية** (`fiscal_periods`): إقفال شهري/ربعي + Triggers
- **القيد المزدوج** (`journal_entries` + `journal_entry_lines`)

### 3️⃣ الفوترة والرسوم [ADVANCED]

- هياكل رسوم + محرك خصومات (مع توجيه GL) + فواتير + تقسيط

### 4️⃣ بوابات الدفع والتسويات [ADVANCED]

- 3 بوابات + معاملات + تسويات بنكية آلية

### 5️⃣ محرك الضرائب المتقدم [ADVANCED]

- `fin_tax_codes`: OUTPUT/INPUT/EXEMPT/ZERO_RATED + حسابات GL

### 6️⃣ سجل التدقيق [ADVANCED]

- `audit_trail`: يسجل كل INSERT/UPDATE/DELETE مع user_id + IP + old/new

### 7️⃣ إدارة الميزانيات [ADVANCED]

- `budgets` + `budget_lines` + View: الميزانية مقابل الفعلي مع مؤشرات الصحة

### 8️⃣ مذكرات الائتمان/الخصم [ADVANCED]

- `credit_debit_notes`: استرداد وتعديل فواتير (WITHDRAWAL/REFUND/PENALTY)

### 9️⃣ القيود المتكررة [ADVANCED]

- `recurring_journal_templates` + سطور: جدولة DAILY → ANNUAL مع auto_post

### 🔟 مراكز التكلفة + ترقيم + أعمار ديون [ADVANCED]

- `cost_centers` (5 مراكز بذرية) + `document_sequences` (6 أنواع) + View aging
